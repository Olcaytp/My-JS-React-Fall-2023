- What is the purpose of the `else if` statement in JavaScript? The "else if" statement allows you to check additional conditions after the initial "if" statement. It provides a way to specify multiple possible outcomes based on different conditions. // Example if (score >= 90) { console.log("You achieved an A.") } else if (score >= 80) { console.log("You achieved a B.") } else if (score >= 70) { console.log("You achieved a C.") } else { console.log("You need to improve your score.") }
